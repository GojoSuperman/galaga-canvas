# Galaga Shooter

Galaga 스타일 세로 스크롤 슈팅 게임. HTML5 canvas + 순수 ES 모듈로 만들었고, **외부 파일이 전혀 없다**:

- 스프라이트: `js/gfx/pixels.js`의 픽셀 맵 + 팔레트를 코드에서 그린다 (이미지 파일 0개).
- 사운드: `js/core/audio.js`의 WebAudio 합성 (오디오 파일 0개).
- 의존성: `npm install`이 필요 없다 (`package.json`의 `dependencies` 자체가 없음).

## 실행 방법

```
npm run serve
```

브라우저에서 `http://127.0.0.1:8080` 접속.

`index.html`을 `file://`로 직접 열면 **동작하지 않는다.** ES 모듈(`import`/`export`)은 CORS 정책상 `file://` 스킴에서 로드가 차단되므로, 반드시 HTTP 서버(`npm run serve` = `python3 -m http.server 8080`)를 통해 열어야 한다.

## 게임 규칙

- 총 10스테이지. 보스는 **5스테이지**(티어 1, 120hp)와 **10스테이지**(티어 2 = 최종 보스, 220hp, 티어1 대비 이동속도 ×1.35 / 탄속 ×1.25 / 스프레드탄 +2)에 등장한다.
- **급강하 중이거나 대형으로 복귀하는 중인 적을 잡으면 점수 2배.** 위험을 감수하고 급강하 타이밍에 맞춰 잡을수록 점수가 오르는 것이 이 게임의 핵심 리스크·리워드 구조다.
- **파워업 4종**
  - **무기 강화**: 레벨 1→2→3, 영구 지속. 단, 피격으로 죽으면 레벨이 1 내려간다 (실드가 없을 때).
  - **실드**: 피격 1회를 흡수하고 사라진다.
  - **연사(rapid)**: 10초간 발사 쿨다운이 절반으로 줄어든다. 버프 중 다시 먹으면 시간이 누적되지 않고 10초로 갱신된다.
  - **스피드**: 10초간 이동속도가 1.4배. 역시 갱신 방식(누적 아님).
- **elite** 적은 플레이어 위치를 조준해서 쏜다. 그 외 모든 일반 적(bee/butterfly/captain)은 플레이어 위치와 무관하게 항상 아래로 직진 사격한다.

## 밸런스를 바꾸고 싶다면 — 어디를 고칠지

| 목적 | 파일 |
|---|---|
| 스테이지 구성, 난이도 전반 (웨이브, 급강하 간격, 적 탄속, 드롭률, 보스 티어 지정 등) | `js/stages/stages.js` — 모든 밸런스 데이터가 여기 모여 있다. 슬롯 충돌·존재하지 않는 적 타입/진입 경로·논보스 스테이지 간 난이도 단조 증가를 테스트가 검증하므로, 오타를 내면 `npm test`가 잡아준다. |
| 캐릭터/적/이펙트 외형 | `js/gfx/pixels.js` — 스프라이트별 픽셀 맵과 팔레트. 여기를 고치면 그림이 바뀐다. |
| 보스 강함 (체력, 이동/탄속 배율, 스프레드 보너스) | `js/game/boss.js` → `BOSS_TIERS` |
| 사운드 (BGM 음량, 멜로디) | `js/core/audio.js` → `BGM_VOLUME` / 멜로디 배열 |
| 특정 스테이지부터 바로 플레이하며 테스트 | `createPlayScene(game, { startStage: N })` — `N`은 0부터 시작하는 스테이지 인덱스. 후반 스테이지를 처음부터 다 깨지 않고 바로 확인할 때 쓴다. |
